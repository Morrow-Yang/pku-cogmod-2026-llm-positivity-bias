# Data

This directory holds the experimental dataset. Per the course's data-and-code submission policy, datasets under 100 MB are bundled directly; larger datasets are documented with regeneration instructions and pointed to their source. Both cases apply here.

```
data/
├── trial_results/          ← In-repo (~16 MB). One JSON per (seed, target) for each pipeline.
└── probes/                 ← Out-of-repo (~315 MB of hidden-state activations).
    └── README.md           ← How to regenerate.
```

## Where the data comes from

This project is unusual in that the experimental data was **generated by querying a large language model**, rather than analyzing a pre-existing human-behavior dataset. Three things constitute "the data":

### 1. The model (the source of all responses)

- **Llama-3.1-8B base** — `meta-llama/Meta-Llama-3.1-8B`, the pretrained-only version, no instruction tuning.
- **Llama-3.1-8B-Instruct** — `meta-llama/Meta-Llama-3.1-8B-Instruct`, the same base after Meta's full RLHF post-training pipeline.

Both are obtained from Hugging Face under Meta's Llama 3.1 license. Model weights are not redistributed here; download separately from the URLs in [`../code/README.md`](../code/README.md).

### 2. The stimuli (the inputs that elicit responses)

Stimuli are defined as **(target × prompt-template)** combinations:

- **Targets** are the entities being rated. They fall into two classes:
  - *Pseudoword targets* (e.g., `adirenia`, `vellinkov`, `brindeck`, `domorath`, `drispalva`, `khantelan`, ...): nonsense words that the model has no real prior information about. **Source: adapted from Lehr et al. (2025)** "Kernels of selfhood: GPT-4o shows humanlike patterns of cognitive dissonance moderated by free choice" (*PNAS Nexus*), which introduced this paradigm for LLM attitude testing. See [`../references/README.md`](../references/README.md) for the full citation.
  - *Real-entity targets* (e.g., `clean_drinking_water`, `che_pact`, ...): familiar concepts and entities used as control items. Curated by the author for this study.
- **Prompt templates** are the linguistic frames in which the model is asked to rate a target. Five templates cover the M0 → M3 staged variants studied in the paper (free-form, system-prompt-only, chat-templated, chat-templated with negativity license). Templates are defined in each pipeline's `config.yaml`.

The complete (target list, template set, replication counts) is reproducible from the YAML configs in [`../code/`](../code/).

### 3. The responses (what we bundle in `trial_results/`)

For each (target × template × seed) the model is queried under greedy decoding (temperature = 0), and its response is parsed into a Likert rating (1–7) plus the raw text. One JSON file per (seed, target) per pipeline. Field schema:

```json
{
  "target": "adirenia",
  "seed": 1,
  "template_idx": 0,
  "stage": "M3",
  "model": "Meta-Llama-3.1-8B-Instruct",
  "prompt": "...",
  "raw_response": "...",
  "parsed_rating": 5,
  "...": "..."
}
```

Note on dedup: greedy decoding made the originally-planned 10 replications per condition deterministically identical, collapsing 7,700 nominal observations to **710 unique observations**. The paper §2.5 documents this and reports all statistics on the deduped set.

## Pipelines and what each generates

| Pipeline | Trial files | Contains |
|---|---|---|
| `stage-behavioral-replication-lehr-2025-free/` | `seed_<S>_target_<T>.json` | Baseline replication of Lehr 2025 phenomenon |
| `stage-base-vs-instruct-positivity-prior/` | per-target JSONs | Base vs Instruct contrast |
| `per-stage-localization-forced-honesty-overshoot/` | per-target JSONs | M0–M3 progression + forced-/control-honesty probes |
| `cognitive-model-comparison-polite-speech-post/` | analysis outputs (model fits, recovery matrices) | Outputs of the 7-model cognitive-model comparison |
| `post-review-extensions/` | per-target JSONs + summary CSVs | β\_L decomposition, pseudoword extension, post-review additions |

## Hidden-state probe data (Study 4) — out of repo

The Study 4 hidden-state activations (660 .npz files, ~315 MB) are not bundled here per the course's >100 MB rule. They are deterministically regenerable from the model + stimuli using `../code/post-review-extensions/extract_hidden_states.py`. See [`probes/README.md`](probes/README.md) for the exact regeneration procedure and expected runtime.

## Provenance summary (citable form)

> **Source models:** Llama-3.1-8B base and Llama-3.1-8B-Instruct, released by Meta AI under the Llama 3.1 license; accessed via Hugging Face.
> **Pseudoword stimuli:** adapted from Lehr, S. A. *et al.* (2025), *PNAS Nexus*, "Kernels of selfhood: GPT-4o shows humanlike patterns of cognitive dissonance moderated by free choice."
> **Real-entity stimuli, prompt-template design, and all experimental procedures:** original to this project (Mingtian Yang, PKU 2026).
> **Generation procedure:** greedy decoding (temperature = 0) over (target × template × seed) cross. Responses parsed into Likert ratings; the unique-observations set (N = 710 after dedup) is the unit of analysis.
